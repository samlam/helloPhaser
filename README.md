===FlappyBird tutorial on Phaser.io + pixi.js

Simple tutorial app using yo generator, bower, and node.js

run the following commands:

* npm install
* bower install
* grunt serve

[Demo available on Heroku](http://flappybirdreborn.herokuapp.com/)
